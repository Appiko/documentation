CustomAppBar class Design Widget

Custom app bar, with option to convert the default, back arrow to down arrow. Example

    Scaffold(
      backgroundColor: Colors.white,
      appBar: CustomAppBar(
        title: "Camera Trigger",
        downArrow: true,
        onDownArrowPressed: () {
          Navigator.pop(context);
        },
      ),
    );
